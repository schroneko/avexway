# 僕と、林真司と小林敏雄。3 人は「グー・チョキ・パーの関係」。

そのときの調査から、林はこんなふうに回想する。「メディアなどでは、松浦はクリエイターとして吹聴されているけど、じつは彼は天才的なマーケッターなんです。あのときの調査は、後々のエイベックスにとって、もの凄く重要だった気がする。正確な数字は分からないけど、レンタル会員の数が 3 万人対 3,000 人だったと思う。ライバル店の会員数は 10 倍だった。友&愛は入会金が 500 円。ライバル店は入会金 0 円。レンタル料は、全部 20 円ずつ向こうが安いとか。店の歴史、在庫量とか、いろんなものが見えてきたなかで、うちの最大のウイークポイントは、入会金の 500 円だという結論に達したんです」

松浦の表現によると、3 人は「グー・チョキ・パーの関係」だという。それは、松浦は林に強くものが言える。しかし、松浦は小林が苦手。けれど、林は小林に好きなことが言える。

小林によると、また表現が変わる。3 人は「ヤジロベエのような関係」だという。松浦と小林が対極に居て、振り子の中心に居るのが、林なのだという。

好きな音楽も考え方も趣味も、まったく違う。互いに我が道を往く。「3 人揃って遊びに行くなんて、考えられない」と林は言う。しかし、この 3 人の出会いは、まさしく宿命的であった。

林は、こう続けた。

「ライバル店の壁をどうやって打破するのか？3 人で考えたわけです。そのときの役割が、いまも同じ役割なんです。小林が財務をやって、僕が従業員のオペレーション、営業をやる。松浦が仕入れ、イベント企画をやる。3 人で決めたことを一斉に 3 人で走る。ずっと、そんなことをやってきた」

全国展開の「友&愛」チェーンには、入会金は一律 500 円という規則があった。しかし、入会金がある以上、ライバル店に勝てる見込みがない。松浦は、「友&愛」本部に乗り込み交渉をもった。

「競争相手が入会金 0 円なので、うちも入会金を 0 円にできないか？」

「全国統一なので、それは絶対に無理だ」「どうしてもダメだというのなら、うちの経営の責任を取ってくれるのか？」

「責任は取れない。入会時の契約でそうなっている。だから、できない」

本部での交渉は、らちが明かなかった。松浦はその場で思いついた方法を提案した。

「じゃあ、キャンペーンを開催して、1 日だけ入会金 0 円で会員を募ってもいいか？」

それで本部を強引に了承させた。松浦がターゲットにしたキャンペーン開催日は、11 月 3 日の「文化の日」だった。
