# まだ流行ってないアーティストの CD を見つけては、“推奨盤”のコメントを付けて卸していた。

「次に、日本でリリースされてない CD で、いい作品を見つけて輸入する。それに、得意の“コメントを付けて御す”という商売を始めた」

輸入盤には、国内盤についている帯やたすきが付いていなかった。松浦は、まだ流行っていないアーティストの CD を見つけては、“推奨盤”のコメントを付けて卸していた。

「それを卸すと、レンタル店からもの凄いオーダーが来た。これだけちゃんと輸入業者をやれるなら、依田さんに CD ワンストップを紹介してもらって、そこから仕入れることで、画期的なシステムを考えようという話になった」

その頃、国内の輸入業者は、輸入盤の在庫を持っていなかった。店からオーダーを受けて発注し、輸入し、通関を終えるまでに早くても 1 週間かかった。どうにかして 1 日でも早く新譜を仕入れることが、輸入業者の勝負だった。

松浦は、新譜の在庫を持ちながら受注があればすぐに出荷できる仕組みをつくりたかった。それには、返品できるシステムが必要だった。例えば、ある新譜を 3,000 枚輸入して、1,000 枚残ったら返品できるという仕組みだ。

米国 CD ワンストップ社との交渉には依田があたった。もともと代表のアラン・メルツァーは日本に CD を卸したかったのだが、日本の輸入業者はレコード時代から付き合いのある輸出業者を大事にしていて、CD ワンストップをまったく相手にしてくれないと、依田に泣きついていた。

そこで松浦は、CD ワンストップの仕入価格から算出した売値で、画期的なシステムを考えようとしていた。

「交渉の結果、毎月、ある一定の返品の枠をもらうことになった。その代わり、保証金 7,000 万円を積んでくれと言われた。当時、そんな金ないじゃないですか。出すのか出さないのか。もの凄く悩んだ挙句の果てに、重野さんに相談したら、『出すべきだ』と言われた。僕は親父に頼んで家を担保に 3,500 万円借りた。鈴木社長はどこからか 3,500 万円を用意した。それはエイベックスにとって最大のターニング・ポイントだった。そのとき、依田さんは 1 円も出してくれなかった」と松浦は依田が保証金の調達に動かなかったことを強調する。
